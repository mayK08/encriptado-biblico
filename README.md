Bible Cipher - Sistema de Cifrado con Versículos Bíblicos

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow)

Un sistema de cifrado/descifrado que utiliza versículos de la Biblia como clave secreta para proteger tus mensajes.

## ✨ Características

- **Cifrado seguro** usando versículos bíblicos
- **Descifrado reversible** con la misma clave
- **Integración con Bible API** para acceso a versículos
- **Interfaz moderna y responsive**
- **100% cliente-side** - sin almacenamiento de datos
- **Multidioma** (versículos en español/inglés)
